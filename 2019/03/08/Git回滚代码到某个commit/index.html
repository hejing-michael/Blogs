Git

Git回滚代码到某个commit

回退命令：

$git reset --hard HEAD^      回退到上个版本  
$git reset --hard HEAD~3     回退三次提交
$git reset --hard commit-id  退到/进到指定commit

强推到远程：

$git push origin HEAD --force
